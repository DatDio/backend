# EmailSieuRe API Documentation

Chào mừng bạn đến với tài liệu API của **EmailSieuRe** - Hệ thống cung cấp mail giá rẻ, uy tín.

---

## Bắt đầu nhanh

### 1. Lấy API Key

Đăng nhập vào [EmailSieuRe](https://emailsieure.com) và lấy API Key tại trang **Cài đặt tài khoản**.

### 2. Sử dụng API

Tất cả API đều yêu cầu tham số `apikey` trong query string:

```
https://emailsieure.com/api/v1/endpoint?apikey=YOUR_API_KEY
```
